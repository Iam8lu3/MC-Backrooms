MC-Backrooms is a procedural Backrooms world generator plugin for Minecraft Java Edition servers running Paper, Spigot, and Bukkit.

The plugin generates vast interconnected Backrooms-inspired environments filled with endless corridors, hidden rooms, dead ends, stairwells, anomalies, hazards, loot, and unique exploration opportunities. Every generated world is designed to feel unfamiliar, unsettling, and nearly endless.

## Features

* Infinite procedural Backrooms generation
* Endless corridors and interconnected room networks
* Multi-floor layouts connected by stairwells
* Flooded sectors
* Boiler rooms and industrial areas
* Random anomalies and environmental hazards
* Furniture, storage areas, and abandoned structures
* Lootable containers
* Monster spawning throughout the world
* Hidden and sealed rooms
* Multiple room entrance styles
* Exploration-focused gameplay
* Survival-friendly generation
* Compatible with Paper, Spigot, and Bukkit

## World Features

Players may encounter:

* Long hallways stretching beyond view
* Random room layouts
* Flooded chambers
* Lava-filled sectors
* Storage areas
* Hidden rooms behind walls
* Sealed rooms containing valuable loot or danger
* Dead ends
* Stairwells connecting different levels
* Random furniture and abandoned equipment
* Hostile mobs wandering the halls
* Rare survivors and villagers

No two generated worlds are exactly alike.

## Installation

1. Download the latest release.
2. Place the plugin `.jar` file inside your server's `plugins` folder.
3. Restart the server.
4. Create a Backrooms world using Multiverse.

```text
/mv create Backrooms normal -g backrooms
```
