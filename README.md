# MC-Backrooms

MC-Backrooms is a procedural Backrooms world generator plugin for Minecraft Java Edition servers running Paper, Spigot, or Bukkit.

The plugin generates vast interconnected environments inspired by Backrooms lore, including endless hallways, branching rooms, dead ends, stairwells, anomalies, and multi-level structures designed for exploration, survival, and horror-themed gameplay.

## Features

* Infinite procedural Backrooms generation
* Connected hallways and room networks
* Multi-floor layouts with stairwells
* Flooded sectors and maintenance areas
* Boiler rooms and industrial zones
* Random anomalies and environmental hazards
* Furniture and abandoned structures
* Monster spawning zones
* Exploration-focused world design
* Compatible with Paper, Spigot, and Bukkit servers

## Installation

1. Download the latest release.
2. Place the plugin `.jar` file into your server's `plugins` folder.
3. Restart the server.
4. Create a Backrooms world using Multiverse or your preferred world management plugin.

## Status

This project is currently under active development. Features, generation algorithms, room layouts, and level systems are continuously being improved.
